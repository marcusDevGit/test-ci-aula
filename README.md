# Projeto de Integração Contínua com GitHub Actions

Este é um projeto de demonstração criado para aprender e praticar Integração Contínua (CI) usando GitHub Actions.

## Sobre o Projeto

O projeto contém uma implementação simples de funções matemáticas em JavaScript, com testes unitários usando Jest. O objetivo principal é demonstrar como configurar e utilizar pipelines de CI para automatizar:

- Execução de testes
- Verificação de qualidade de código
- Processo de build
- Deploy automatizado

## Tecnologias Utilizadas

- JavaScript (ES Modules)
- Jest (Framework de Testes)
- GitHub Actions (CI/CD)

## Como Executar

Para rodar os testes localmente:

```bash
npm install
npm test
```
